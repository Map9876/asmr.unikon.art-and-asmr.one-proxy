# asmr.unikon.art-and-asmr.one-proxy
一个.zip压缩包格式的浏览器拓展，用于网络不好时候的通过 cloudflare workers按 https://example.dpdns.org/https://api.asmr-200.com
直连。适用于kikoeru系网站。

注 :https://example.dpdns.org/ 是workers.dev绑定的域名，cloudflare workers反代是https://github.com/ymyuuu/Cloudflare-Workers-Proxy 项目部署的，其作用是 workers.dev/https://baidu.com 等同于显示  https://baidu.com
